# 简介

用于提高Cesium可用性的插件库。

# 目录

  • PositionPicker (opens new window)：用于鼠标左键点击拾取经度、纬度和高度的工具类。

  • Cesium3DTilesLoader (opens new window)：用于加载并配置3DTiles的工具类。

  • Tooltip (opens new window)：用于创建跟随鼠标移动的Tooltip工具类。

  • exportSceneAsImage (opens new window)：用于将场景导出为图片（.png）的方法。

  • FloodAnalysis (opens new window)：用于模拟淹没分析的工具类。

  • RotateAroundPoint (opens new window)：用于相机绕点旋转的工具类。

  • WaterMask (opens new window)：用于创建动态水面的工具类。

  • HTMLOverlay (opens new window)：用于将HTML元素放在Cesium canvas上的工具类。
